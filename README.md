# concert-ticketing-service

## 프로젝트 소개
대규모 트래픽을 처리할 수 있는 고성능 콘서트 티켓 예매 시스템입니다.

## 기술 스택
- **Backend**: Spring Boot 3, Java 21, JPA, MySQL, Redis
- **Frontend**: Nuxt 4, Pinia, Tailwind CSS
